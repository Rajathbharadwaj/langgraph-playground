# 🎯 LangGraph Visual Builder

Build AI agents visually with LangGraph. Design workflows, drag-and-drop nodes, export to Python.

**[Try it free →](https://rajathbharadwaj.github.io/langgraph-playground/)**

![LangGraph Builder Screenshot](https://rajathbharadwaj.github.io/langgraph-playground/screenshot.png)

## Features

- 🤖 **LLM Nodes** - Connect Claude, GPT-4, or any model
- 🔧 **Tool Nodes** - API calls, web search, code execution
- 🔀 **Conditional Routing** - Branch based on LLM output
- 👤 **Human-in-Loop** - Pause for human approval
- 💾 **Memory Nodes** - Persistent context across turns
- 📊 **Subgraphs** - Nested agent workflows

## Quick Start

1. Visit [langgraph-playground](https://rajathbharadwaj.github.io/langgraph-playground/)
2. Drag nodes onto the canvas
3. Connect them to build your workflow
4. Export to Python code
